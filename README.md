# Casa VIP Móveis - Landing Page Premium

Projeto de landing page desenvolvido para a **Casa VIP Móveis**, com foco em sofisticação e experiência do usuário (UX) para o mercado de Canela e Gramado/RS.

## 🚀 Tecnologias Utilizadas
* **Tailwind CSS**: Estilização rápida e responsiva.
* **Google Fonts**: Tipografia Playfair Display e Inter para tom luxuoso.
* **Bootstrap Icons**: Iconografia minimalista.
* **JavaScript (Vanilla)**: Integração direta com API do WhatsApp para leads.

## 📁 Estrutura do Projeto
O projeto utiliza caminhos relativos para navegação entre as linhas de produtos (Estar, Jantar e Descanso), garantindo que a estrutura seja independente de servidor.
